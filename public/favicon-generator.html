Favicon Generator

This file can be used to generate PNG favicons from the SVG favicon.

To generate the PNG files:

  1. Open this file in a browser
  2. Use browser dev tools to convert the SVG to PNG
  3. Save as favicon-16x16.png, favicon-32x32.png, etc.
